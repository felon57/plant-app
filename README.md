# Plant App (planet_app)

یک اپلیکیشن Flutter برای نمایش و اسکن اطلاعات گیاهان.

شروع سریع:
- نصب Flutter SDK
- اجرا: flutter pub get
- اجرا محلی: flutter run

ساختار:
- lib/: کد منبع (screens, widgets, models)
- assets/images/: تصاویر مورد استفاده

توجه: صفحه اسکن در lib/screens/scan_page.dart از بسته mobile_scanner استفاده می‌کند.
فایل تنظیم lint (analysis_options.yaml) اضافه شده است.
